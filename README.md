# Pomodoro Timer

Minimalist pomodoro timer with optional stream/water ambience. The stream sound plays while the timer runs and stops automatically when time is up.

## Run

Open `index.html` in a browser (double-click or `open index.html`).

## Stream sound

- The page uses a fallback stream sound from the web. For best results (and offline use), add your own file named **`stream.mp3`** in this folder — any flowing water/stream MP3 will work. The timer will use it first if present.

## Usage

- **Start** — begins the countdown and starts the stream sound.
- **Stop** — resets and stops the timer and sound.
- Presets (**25 min**, **15 min**, **5 min**) set the duration; then press Start.
